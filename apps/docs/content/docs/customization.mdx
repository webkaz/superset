## Notification Sounds

Get audio notifications when agents finish or terminals exit.

Settings → Notifications → Choose a sound

## Project Colors

Color-code your projects in the sidebar for quick identification.

Click project settings → Choose a color

## Default Editor

Set which editor opens when you press ⌘O.

Settings → Editor → Select Cursor, VS Code, or other

## Terminal Presets

Save terminal configurations (commands, environment) as presets for quick access.

Settings → Terminal → Manage Presets

## Settings Sync

Your settings sync across devices when signed in.
