## Workspace

| Action | Shortcut |
|--------|----------|
| New Workspace | ⌘N |
| Quick Create Workspace | ⌘⇧N |
| Open Project | ⌘⇧O |
| Switch to Workspace 1-9 | ⌘1-9 |
| Previous Workspace | ⌘⌥↑ |
| Next Workspace | ⌘⌥↓ |

## Layout

| Action | Shortcut |
|--------|----------|
| Toggle Changes Tab | ⌘L |
| Toggle Expand Sidebar | ⌘⇧L |
| Toggle Workspaces Sidebar | ⌘B |
| Split Right | ⌘D |
| Split Down | ⌘⇧D |
| Split Pane Auto | ⌘E |
| Close Pane | ⌘W |

## Terminal

| Action | Shortcut |
|--------|----------|
| New Tab | ⌘T |
| Close Terminal | ⌘W |
| Clear Terminal | ⌘K |
| Find in Terminal | ⌘F |
| Scroll to Bottom | ⌘⇧↓ |
| Previous Tab | ⌘⌥← |
| Next Tab | ⌘⌥→ |
| Previous Pane | ⌘⇧← |
| Next Pane | ⌘⇧→ |
| Open Preset 1-9 | ⌘⇧1-9 |

## Window

| Action | Shortcut |
|--------|----------|
| Open in App | ⌘O |
| Copy Path | ⌘⇧C |
| Close Window | ⌘⇧W |
| Show Keyboard Shortcuts | ⌘/ |

## Customize

Open Settings → Keyboard Shortcuts to customize any hotkey.

## Import/Export

Share your hotkey configuration:
- **Export** - Save your shortcuts to a file
- **Import** - Load shortcuts from a file
